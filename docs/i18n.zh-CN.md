# 国际化（i18n）

[English](i18n.md) · **简体中文**

## 规则

1. **主语言为英文。** 权威路径保持 `.md` 文件名（如 `README.md`、`skills/pro-generation.md`）。
2. **中文为并行副版。** 同目录、后缀 `.zh-CN.md`（如 `README.zh-CN.md`）。
3. **Agent 默认只读英文主版。** 除非用户明确要求中文输出，否则不要用 `.zh-CN.md` 作为技能/流程契约。
4. 双语文档页眉宜加语言切换，例如：

   `[English](foo.md) · **简体中文**`

5. 中英文章节结构尽量对齐，方便 diff。
6. 新增文档：先英文；可行时同变更补 `.zh-CN.md`。

## 命名

| 类型 | 英文（权威） | 中文 |
|------|--------------|------|
| 默认 | `name.md` | `name.zh-CN.md` |
| 根 README | `README.md` | `README.zh-CN.md` |
| Agent 入口 | `AGENTS.md` | `AGENTS.zh-CN.md` |

## 维护

- 行为变更先改 **英文** skill / workflow。
- 面向用户的变更尽量同 PR 更新中文副版。
- 若中英文不一致，以英文为准。
